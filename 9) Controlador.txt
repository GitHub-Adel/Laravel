sirve de intermediario entre la vista y el modelo.

flujo:
-----------------------
1) el usuario ejecuta la solicitud a traves de una url
2) le pide los datos al modelo
3) procesa los datos si se requiere.
4) le pasa el modelo a la vista.
5) le devuelve la vista al usuario.

Crear un controlador para Cliente:
----------------------------
1) abrir carpeta app
2) abrir carpeta Http
3) abrir carpeta Controllers
4) crear  archivo ClienteController.php
5) agregar el siguiente contenido

<?php

namespace App\Http\Controllers;

class ClienteController extends Controller
{
    public function Crear(){
        return "Creando cliente";
    };

    public function Editar($id){
        return "editando cliente " . $id;
    };
}


Nota:
------------------------
<?php	: inicio de codigo php
namespace	: especifica la carpeta en la que se encuentra la clase o el controlador.
extends 	: herencia
